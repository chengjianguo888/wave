# 如何运行 (中文说明)

## 🎯 这是什么？

这是一个**纯Python实现**的波浪检测系统，使用Streamlit框架。
你可以直接在PyCharm中运行，无需安装Node.js、npm或Docker！

## ⚡ 最快启动方式

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 运行应用
```bash
python run.py
```

### 3. 打开浏览器
访问: http://localhost:8501

就这么简单！✅

## 📱 在PyCharm中运行

### 方法1: 直接运行（最简单）
1. 在PyCharm中打开项目
2. 找到并打开 `run.py` 文件
3. 点击右上角的绿色运行按钮 ▶️
4. 浏览器会自动打开应用

### 方法2: 使用Terminal
在PyCharm下方的Terminal中输入：
```bash
python run.py
```

### 方法3: 创建运行配置
1. 菜单栏: Run → Edit Configurations...
2. 点击 + → Python
3. Script path: 选择 `run.py`
4. 点击 OK
5. 点击运行按钮

## 🎨 功能介绍

### 登录和注册
- 首次使用需要注册账户
- 输入用户名、邮箱、密码即可

### 波浪检测
1. 上传图像文件（JPG或PNG）
2. 选择检测算法（4种可选）
3. 调整灵敏度（0.0-1.0）
4. 点击"开始检测"按钮
5. 查看检测结果

### 检测算法说明
- **边缘检测**: 快速、适合清晰边界
- **频率分析**: 适合周期性波浪
- **光流法**: 适合运动检测
- **AI检测**: 智能识别复杂模式

### 历史记录
- 查看所有检测历史
- 导出为CSV或JSON格式

## ⚙️ 系统要求

- Python 3.8 或更高
- Windows, macOS 或 Linux
- 2GB RAM (推荐4GB)

## 🔧 常见问题

### 安装依赖失败？
```bash
pip install --upgrade pip
pip install -r requirements.txt
```

### 端口被占用？
编辑 `run.py`，修改端口号：
```python
'--server.port', '8502',  # 改为其他端口
```

### OpenCV安装失败？
```bash
pip uninstall opencv-python
pip install opencv-python==4.9.0.80
```
