## 🚀 如何运行（3步）

### 第一步: 安装依赖
在Terminal或PyCharm的Terminal中运行：
```bash
pip install -r requirements.txt
```

**如果安装慢，使用国内镜像:**
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 第二步: 验证环境（可选但推荐）
```bash
python verify_setup.py
```

这会检查所有依赖是否正确安装。

### 第三步: 运行应用

**在PyCharm中:**
1. 打开 `run.py` 文件
2. 点击右上角的绿色运行按钮 ▶️
3. 或按 `Shift+F10`

**在Terminal中:**
```bash
python run.py
```

**或直接运行Streamlit:**
```bash
streamlit run app.py
```

### 访问应用
浏览器会自动打开: **http://localhost:8501**

## 🎨 功能说明

### 1. 登录和注册
- 首次使用需要在"注册 (Register)"标签注册账户
- 然后在"登录 (Login)"标签登录

### 2. 波浪检测
- 在"波浪检测"页面上传图像
- 选择4种算法之一:
  - **边缘检测** - 适合清晰边界
  - **频率分析** - 适合周期性模式
  - **光流法** - 适合运动分析
  - **AI检测** - 智能识别
- 调整灵敏度（0.0-1.0）
- 点击"开始检测"

### 3. 查看结果
- 波浪数量、振幅、频率
- 置信度评分
- 详细数据表格
- 特征图表

### 4. 历史记录
- 在"检测历史"页面查看所有记录
- 可以导出为CSV或JSON
