# Visual Wave Detection System - User Guide

## 功能详解

### 1. 登录系统
系统提供安全的用户认证功能:
- **注册**: 创建新账户需要用户名、邮箱和密码
- **登录**: 使用用户名和密码登录
- **会话管理**: 基于JWT token，24小时有效期

### 2. 波浪检测功能

#### 支持的图像格式
- PNG, JPG, JPEG, GIF, BMP
- 最大文件大小: 50MB

#### 四种检测算法

##### Edge Detection (边缘检测)
- **最佳用途**: 清晰的波浪边界，高对比度图像
- **工作原理**:
  - 使用Canny边缘检测算法
  - 识别强度变化的边界
  - 提取波浪轮廓
- **参数**: 灵敏度控制边缘检测阈值
  - 低灵敏度(0-0.3): 只检测明显边缘
  - 中灵敏度(0.3-0.7): 平衡检测
  - 高灵敏度(0.7-1.0): 检测微弱边缘

##### Frequency Analysis (频率分析)
- **最佳用途**: 周期性波浪模式
- **工作原理**:
  - 使用快速傅里叶变换(FFT)
  - 分析频域特征
  - 识别周期性模式
- **参数**: 灵敏度控制频率峰值检测
- **输出**: 包含频率范围信息

##### Optical Flow (光流法)
- **最佳用途**: 动态波浪，运动分析
- **工作原理**:
  - 计算像素运动场
  - 识别运动模式
  - 提取波浪运动特征
- **参数**: 灵敏度控制运动阈值
- **特点**: 适合视频帧分析

##### AI Detection (AI检测)
- **最佳用途**: 复杂场景，多种波浪类型
- **工作原理**:
  - 使用高级图像处理
  - 形态学分析
  - 智能特征提取
- **参数**: 灵敏度控制置信度阈值
- **优点**: 最智能，适应性最强

### 3. 结果解读

#### 检测指标
- **Wave Count (波浪数量)**: 检测到的波浪个数
- **Average Amplitude (平均振幅)**: 波浪的平均高度（像素）
- **Average Frequency (平均频率)**: 波浪的空间频率
- **Confidence Score (置信度)**: 检测结果的可信度 (0-100%)

#### 可视化功能
- **标注图像**: 在原图上标注检测到的波浪位置
- **振幅分布图**: 显示各个波浪的振幅变化
- **详细数据表**: 列出每个波浪的具体参数

### 4. 历史记录管理
- 自动保存所有检测结果
- 按时间倒序排列
- 分页浏览(每页10条)
- 支持CSV格式导出

### 5. 数据导出
导出的CSV文件包含:
- Wave ID: 波浪编号
- Position X/Y: 波浪位置坐标
- Amplitude: 振幅
- Wavelength: 波长
- Area: 面积
- Confidence: 置信度

## 使用技巧

### 选择合适的算法
1. **清晰的海浪照片**: 使用 Edge Detection
2. **规律的水波纹**: 使用 Frequency Analysis
3. **视频截图**: 使用 Optical Flow
4. **复杂场景**: 使用 AI Detection

### 调整灵敏度
- 检测结果太少: 提高灵敏度
- 检测到太多噪声: 降低灵敏度
- 建议从50%开始，根据结果调整

### 图像预处理建议
- 使用高清图像获得更好结果
- 确保波浪特征清晰可见
- 避免过度曝光或欠曝光
- 图像中波浪占比应大于10%

## 常见问题

### Q: 检测不到波浪怎么办？
A: 尝试:
1. 提高灵敏度参数
2. 切换不同的检测算法
3. 检查图像质量和清晰度

### Q: 检测结果不准确？
A: 建议:
1. 根据场景选择合适的算法
2. 调整灵敏度参数
3. 使用高质量图像

### Q: 如何导出所有历史数据？
A: 在历史记录页面，逐个点击"Export"按钮导出每条记录的CSV文件

### Q: 系统支持视频检测吗？
A: 当前版本支持从视频中提取帧进行检测，未来版本将支持完整视频流分析
