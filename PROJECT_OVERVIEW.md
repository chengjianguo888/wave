# Visual Wave Detection System - Project Overview

## 项目简介

视觉波浪检测系统是一个创新的计算机视觉应用，专门用于自动检测和分析图像中的波浪模式。系统集成了多种先进的图像处理算法，提供直观的Web界面和完整的用户管理功能。

## 核心创新点

### 1. 多算法融合架构
系统独特地集成了四种不同的波浪检测算法:
- **经典方法**: 边缘检测和频率分析
- **现代方法**: 光流法和AI智能检测
- **自适应选择**: 用户可根据场景选择最佳算法

### 2. 实时可视化系统
- 波浪边界实时标注
- 交互式统计图表
- 详细的参数分析
- 支持大规模数据处理

### 3. 企业级架构
- 完整的用户认证系统
- 数据持久化存储
- RESTful API设计
- Docker容器化部署

## 技术亮点

### 后端技术
1. **Flask Web框架**: 轻量级、灵活的Python Web框架
2. **OpenCV**: 工业级图像处理库
3. **NumPy/SciPy**: 高性能科学计算
4. **SQLite**: 零配置的嵌入式数据库
5. **JWT认证**: 安全的无状态认证

### 前端技术
1. **React 18**: 现代化的UI框架
2. **Vite**: 极速的构建工具
3. **Chart.js**: 强大的数据可视化
4. **响应式设计**: 适配各种屏幕尺寸
5. **HTML5 Canvas**: 高性能图像渲染

### 算法技术
1. **Canny边缘检测**: 最优边缘检测算法
2. **FFT频率分析**: 频域信号处理
3. **Farneback光流**: Dense光流估计
4. **形态学分析**: 数学形态学处理
